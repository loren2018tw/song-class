## MODIFIED Requirements

### Requirement: 小白板元件必須使用固定內部畫布尺寸並可縮放顯示

小白板元件 MUST 以 1280×720 px 作為固定內部畫布座標系，並以「背景圖層 + 透明塗鴉層」兩層畫布在相同座標下疊合顯示。

#### Scenario: 元件在不同容器大小下顯示

- **WHEN** 小白板元件被放入不同寬高的顯示區域
- **THEN** 系統 MUST 以相同的 1280×720 內部座標繪製背景層與塗鴉層
- **AND** 系統 MUST 將兩層畫布以相同縮放比例呈現

#### Scenario: 同一筆畫在縮放後仍維持相同相對位置

- **WHEN** 內容已在固定畫布座標系中完成塗繪
- **THEN** 系統 MUST 在任何縮放比例下維持筆畫的相對位置與比例

### Requirement: 小白板元件必須支援滑鼠與觸控塗繪

小白板元件 MUST 支援滑鼠與觸控輸入進行連續塗繪，並提供畫筆顏色、畫筆粗細、真正橡皮擦與清除塗鴉功能。

#### Scenario: 使用滑鼠繪圖

- **WHEN** 使用者按住滑鼠並在畫布上拖曳
- **THEN** 系統 MUST 以目前畫筆顏色與粗細在塗鴉層繪製連續線段

#### Scenario: 使用觸控繪圖

- **WHEN** 使用者以觸控筆或手指在畫布上拖曳
- **THEN** 系統 MUST 以與滑鼠相同的方式在塗鴉層繪製連續線段

#### Scenario: 切換橡皮擦

- **WHEN** 使用者點選橡皮擦按鈕
- **THEN** 系統 MUST 進入可擦除已繪製筆跡的模式
- **AND** 系統 MUST 僅擦除塗鴉層像素，不得覆蓋或改變背景層內容

#### Scenario: 清除畫布

- **WHEN** 使用者點選清除畫布功能
- **THEN** 系統 MUST 清空目前所有塗鴉內容
- **AND** 系統 MUST 保留目前背景層內容不變

### Requirement: 小白板元件必須輸出可重現的 JSON 塗繪資料

小白板元件 MUST 將塗鴉內容整理為可序列化 JSON，並透過事件傳送給父元件，以便後續傳送到遠端元件或重新還原內容；背景資料 MUST 與塗鴉筆畫資料分離表示。

#### Scenario: 塗繪內容更新時向父元件回傳資料

- **WHEN** 使用者完成一段塗繪、切換工具或清除畫布
- **THEN** 小白板元件 MUST 透過事件回傳最新 JSON 塗繪資料給父元件

#### Scenario: 背景資料更新時向父元件回傳資料

- **WHEN** 父元件變更白板背景圖片資料並由元件套用成功
- **THEN** 小白板元件 MUST 在可重現資料中保留目前背景資訊
- **AND** 背景資訊 MUST 與塗鴉筆畫分開儲存

#### Scenario: 遠端元件還原塗繪內容

- **WHEN** 父元件將相同 JSON 塗繪資料提供給另一個小白板元件實例
- **THEN** 該元件 MUST 還原出相同的背景內容與塗鴉內容

#### Scenario: JSON 內容可供後續傳送

- **WHEN** 父元件取得小白板輸出的 JSON
- **THEN** 該資料 MUST 可直接作為後續同步或儲存的輸入

## ADDED Requirements

### Requirement: 教師背景選擇必須同步到白板背景層

教師端頁面 MUST 在右側「教師背景」下拉選單值變更時，將對應圖片資料傳入小白板元件並更新背景層。

#### Scenario: 教師切換背景選項

- **WHEN** 教師在右側「教師背景」下拉選單改選另一個背景項目
- **THEN** 系統 MUST 將所選背景圖片資料傳給小白板元件
- **AND** 小白板元件 MUST 立即重繪背景層

#### Scenario: 教師清空背景選項

- **WHEN** 教師將背景選項切回無背景
- **THEN** 系統 MUST 清空白板背景層圖片
- **AND** 系統 MUST 保留既有塗鴉內容
