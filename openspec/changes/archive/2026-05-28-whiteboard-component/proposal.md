## Why

目前教師端只有學生清單與連線資訊，缺少可直接在課堂中展示與互動的即時塗繪區域。新增小白板元件後，教師可以在同一頁面快速切換到白板教學，並把塗繪內容以結構化資料傳送到遠端元件重現。

## What Changes

- 在教師端頁面新增「小白版」功能按鈕，讓教師可切換至白板主畫面。
- 調整教師端頁面布局，將功能按鈕列放在上方，學生清單放在左下方，小白板主畫面放在右下方。
- 新增一個可重用的 Vue 小白板元件，支援固定畫布尺寸、縮放顯示與即時塗繪。
- 小白板元件提供顏色、粗細、橡皮擦與清除畫布功能，方便課堂快速標註。
- 小白板元件會輸出可序列化的 JSON 塗繪紀錄，並透過事件回傳給父元件，供後續同步到遠端元件或持久化使用。

## Capabilities

### New Capabilities

- `teacher-whiteboard`: 教師端小白板切換、版面配置、塗繪操作與可序列化輸出。

### Modified Capabilities

- 無。

## Impact

- 影響教師端 Vue 視圖與共用元件結構。
- 可能需要新增白板相關型別、事件與塗繪資料格式。
- 後續若要同步到遠端瀏覽器或儲存課堂內容，會依賴這次產生的 JSON 事件資料。
